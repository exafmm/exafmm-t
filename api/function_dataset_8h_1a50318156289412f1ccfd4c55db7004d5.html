template<typename T>
Bodies<T> exafmm_t::init_targets(int numBodies, const char *distribution, int seed)¶

Generate targets with various distributions.

Return

Bodies Vector of targets.

Template Parameters

  • T: Target’s value type (real or complex).

Parameters

  • numBodies: Number of bodies.

  • distribution: Type of distribution: ‘c’ for cube, ‘s’ for sphere, ‘p’ for plummer.

  • seed: Seed of pseudorandom number generator.
